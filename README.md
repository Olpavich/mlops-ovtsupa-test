Setup Instructions
1. Clone the repository:
    ```bash
    git clone git@github.com:Olpavich/mlops-ovtsupa-test.git
    cd mlops-ovtsupa-test
    ```
2. Create a project:
    ```bash
    uv python pin 3.14.4
    uv init
    uv venv
    uv add --requirements requirements.txt
    ```
